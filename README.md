## 比大小
* 名稱：撲克牌比大小
* 功能：
* 按 start 啟動計時器，計時器會以每0.01秒(對，0.01秒)用亂數切換poker1.jpg~poker13.jpg 撲克牌圖。
* 按 stop 顯示點數及對應鋪克牌圖。
